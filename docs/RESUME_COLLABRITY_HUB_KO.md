# 이력서 프로젝트 경력 기술

---

## Collabrity Hub Core/PMS 시스템 개발

**기간**: 2026.01 ~ 2026.03
**역할**: Full-Stack 개발자 (React, Spring Boot)
**기술 스택**: React, Spring Boot, Java, Scheduler

---

### 프로젝트 개요

건설·엔지니어링 프로젝트 관리에 특화된 협업 플랫폼(Collabrity Hub)의 Core 모듈과 PMS(Project Management System) 모듈을 설계 및 개발. 조직 계층 관리, 프로젝트 결재 프로세스, 타스크 의존 관계 관리 등 핵심 기능을 구현하였으며, 자동화된 스케줄러 서버를 통해 타스크 진행 처리를 자동화.

---

### 주요 담당 업무

#### [Frontend - React]

- **CORE 모듈** – 체크인/체크아웃 UI 컴포넌트 설계 및 구현
  - 사용자의 업무 시작·종료 흐름을 직관적으로 처리하는 UI 컴포넌트 설계
  - 상태 전환 로직과 연계된 재사용 가능한 컴포넌트 구조 구현

- **PMS 모듈** – 프로젝트 결재선 구성 및 관리 패널 UI 구현
  - 결재선 단계별 승인자 설정 및 순서 조정이 가능한 관리 패널 UI 개발
  - 드래그앤드롭 기반의 결재선 편집 인터페이스 구현

- **PMS 모듈** – 타스크 선후행 의존 관계 편집 UI 구현
  - 타스크 간 선행/후행 의존 관계를 시각적으로 편집할 수 있는 UI 구현
  - 사이클 감지 및 유효성 검증 로직을 포함한 의존 관계 그래프 편집 기능 개발

- **PMS 모듈** – 분야(Discipline) 관리 어드민 대시보드 페이지 구현
  - 프로젝트 내 분야별 현황 및 인원 배치를 한눈에 확인할 수 있는 어드민 대시보드 개발
  - 데이터 시각화 및 필터링 기능이 포함된 관리 화면 구현

#### [Backend - Spring Boot]

- **CORE 모듈** – 조직 계층 구조 관리 API 설계 및 구현
  - 부서·팀·직책 등 다단계 조직 계층을 관리하는 RESTful API 설계 및 구현
  - 계층 구조 조회 최적화 및 재귀적 데이터 처리 로직 구현

- **PMS 모듈** – 타스크 분야·역할 할당 및 변경 이력 관리 로직 구현
  - 타스크에 분야 및 역할을 할당하고, 변경 시 이력을 자동으로 기록하는 로직 구현
  - 감사(Audit) 로그 설계를 통해 변경 추적 가능한 데이터 관리 체계 수립

- **PMS 모듈** – 프로젝트 결재선 등록·승인·반려 프로세스 구현
  - 결재선 등록부터 최종 승인/반려까지 전체 워크플로우 백엔드 로직 구현
  - 결재 상태 전이 관리 및 알림 연동 처리 구현

- **스케줄러 서버** – 프로젝트 타스크 자동 진행 처리 스케줄러 서버 구현
  - 프로젝트 타스크의 일정 기반 자동 상태 전환을 처리하는 독립적인 스케줄러 서버 개발
  - Spring Scheduler를 활용한 주기적 배치 처리 및 예외 상황 핸들링 로직 구현

---

### 성과 및 기여

- React 기반 복잡한 UI(결재선 편집, 의존 관계 그래프 등)를 컴포넌트 단위로 설계하여 재사용성과 유지보수성 향상
- Spring Boot 기반 API 설계 시 도메인 모델과 계층 구조를 명확히 분리하여 확장 가능한 아키텍처 구현
- 스케줄러 서버를 별도 모듈로 분리하여 타스크 자동화 처리의 독립성 및 안정성 확보
- 결재 워크플로우 설계를 통해 비즈니스 프로세스의 일관성 및 추적 가능성 확보
